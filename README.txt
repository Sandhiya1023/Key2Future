LOGIN SYSTEM - SETUP INSTRUCTIONS
=====================================

QUICK START:
1. Extract this zip file to your Desktop
2. Double-click "index.html" to open in browser
3. Login with: admin/admin123 or student1/student1

BETTER METHOD (Recommended):
1. Open Command Prompt
2. Type: cd Desktop\new-login-system
3. Type: python -m http.server 8000
